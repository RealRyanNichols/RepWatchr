# RepWatchr Sponsorship Boundaries

RepWatchr may eventually sell clearly labeled civic education sponsorships, organization accounts, dashboards, data products, and monitoring packages.

RepWatchr must not create hidden sponsored content, pay-to-change public records, pay-to-remove correction history, or private political-interest profile sales.

## Hard Rules

- Sponsored content must be clearly labeled.
- Sponsorship must not change source labels.
- Sponsorship must not change correction workflows.
- Sponsorship must not change public-record standards.
- Sponsorship must not imply RepWatchr endorses a candidate, donor, PAC, party, agency, or official.
- Sponsorship must not be sold as a way to bury, hide, suppress, or manipulate public records.
- Sponsorship must not create private targeting dossiers from sensitive political-interest profiles.
- Sponsorship must not promise political outcomes, press coverage, votes, fundraising results, investigations, prosecutions, or removals from office.

## Allowed Future Sponsorship Categories

These are allowed only if clearly labeled and editorial/source standards remain separate:

- civic education explainers
- public-records literacy content
- open-meeting education
- source packet education
- public-record request education
- nonpartisan voter process education
- transparency tool sponsorship

## Disallowed Public Claims

Do not publish copy that says or implies:

- guaranteed returns
- guaranteed political results
- investment terms are being offered
- sponsored content is independent when it is not
- a sponsor controls RepWatchr coverage
- a sponsor can remove or suppress records
- a sponsor can buy a better score
- RepWatchr sells private political-interest profiles

## Sponsor Labeling

If sponsorship is enabled later, visible labels should be plain:

- `Sponsored civic education`
- `Sponsor-supported public-records guide`
- `Paid sponsor label`

Avoid vague labels such as:

- `Partner content`
- `Featured`
- `Presented by`

Those labels can hide the paid nature of the content.

## Separation From Profiles

Sponsored civic education must stay separate from official profile evidence.

Profile pages should continue to prioritize:

- public source links
- source labels
- correction requests
- confidence labels
- what is confirmed
- what needs source review

No sponsor should be allowed to:

- edit a profile score
- hide a source
- change a red flag label
- override a correction
- alter a public relationship graph
- remove a funding record

## Admin Review

Any future sponsor or partner account should include:

- sponsor/account name
- contact email
- sponsorship type
- public label language
- start and end dates
- admin notes
- audit trail

Sponsored content should be reviewable in admin before it is public.

## Default State

Sponsorship is an interest and pipeline category only. It is not enabled as a public paid product in the current implementation.
