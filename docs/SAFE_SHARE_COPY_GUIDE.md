# Safe Share Copy Guide

RepWatchr should make people share the receipt, not just the outrage.

Every share line should help a reader do one of these things:

- open a public record
- inspect a source trail
- submit a missing source
- ask a public question
- request a correction
- watch an official, race, public body, or record
- build a source packet

## Core Prompt

Primary:

```text
Help attach the receipt.
```

Secondary:

```text
Send this to someone who follows this official, race, or public body.
```

## Approved Templates

### Source Gap Share

```text
RepWatchr is looking for a public source for [subject] on [topic]. Add a source here: [URL].
```

### Profile Share

```text
Watch public-record updates for [official/entity] on RepWatchr: [URL].
```

### Public Question Share

```text
Public question for [subject]: [question]. Source/context: [URL].
```

### Race Share

```text
Follow public-record updates for [race] here: [URL].
```

### Packet Share

```text
I built a public source packet for [subject]. Review or add sources here: [URL].
```

### Correction Share

```text
See something wrong on this public profile? Request a correction here: [URL].
```

## Banned Share Framing

Do not write share copy that says or implies:

- destroy
- expose them
- take them down
- go after
- target
- harass
- guilty
- criminal

If a public source uses strong language, RepWatchr can summarize what the source says with a label, but default share copy should avoid those words.

## Safer Replacements

Instead of attack language, use:

- `public source`
- `public record`
- `source trail`
- `record gap`
- `public question`
- `correction request`
- `watch updates`
- `add the missing source`
- `inspect the receipt`

## Examples

Safe:

```text
Watch public-record updates for the Texas Senate race on RepWatchr: https://www.repwatchr.com/elections/texas
```

Safe:

```text
Public question for this board: where are the approved meeting minutes posted? Source/context: https://www.repwatchr.com/school-boards
```

Unsafe:

```text
Letâ€™s expose them and take them down.
```

Why unsafe:

It sounds like a harassment campaign instead of a public-record review.

## Admin Review

Admin campaign copy should be reviewed before activation. The admin manager checks for risky phrases and blocks unsafe default campaign text in the server route.

Good admin checklist:

- Does the line point to a public record or source gap?
- Does it avoid legal conclusions?
- Does it avoid harassment language?
- Does it ask for a useful next action?
- Does it invite correction if the page is wrong?
