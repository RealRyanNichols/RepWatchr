# RepWatchr School Board Buildout Plan

## Rule

A school-board member is not counted as a completed public profile until the profile has a sourced roster record, source links, term or seat data when available, photo status, public-contact path, evidence ledger, praise/concern review, and claim status.

## Analytics Labels

| Label | Meaning |
| --- | --- |
| Profiles live | Board-member records currently loaded into RepWatchr from sourced files or official roster stubs. |
| District files live | District profile pages currently available on the public site. |
| Source-backed districts live | District pages with official source links, district roster data, or sourced candidate/member files. |
| Directory target | A state or national source-directory import target, not a completed board-member profile count. |
| Trustee target | A sourced estimate or benchmark for elected trustee/member profiles to build. |

## Completion Gates

1. Directory record imported from NCES, TEA/AskTED, or a state education agency.
2. District profile shell created with county, state, official website, phone, superintendent when sourced, board size when sourced, and source links.
3. Board roster pulled from district page, election filing, minutes, or state/county source.
4. Member profile created with name, role, seat/place, term, district, county, and status.
5. Photo attached from official/campaign/claimed approved source, or branded initials placeholder used.
6. Evidence ledger added for each score-moving claim.
7. Praise report reviewed for verified good actions.
8. Concern/red-flag review completed only from source-backed records.
9. Political-lean model marked as sourced, inferred, insufficient evidence, or not applicable.
10. Claim profile button active; claimed edits remain separate from RepWatchr facts.
11. Public question/comment module enabled with verified, parent, official, journalist, voter, and anonymous weighting labels.
12. Admin review status set to complete, needs review, stub, or initial dossier.

## Rollout Order

1. Finish all currently loaded East Texas district files and member profiles.
2. Import all Texas district directory records from TEA/AskTED and NCES.
3. Verify every Texas board roster from official district, election, county, and minutes sources.
4. Add Texas member profiles, photos/placeholders, claim buttons, and public question tools.
5. Expand the same import and verification pipeline to every state and District of Columbia.

## Source Priorities

1. District official board pages.
2. State education agency directories.
3. County election offices and candidate filings.
4. Board agendas, minutes, videos, and policy pages.
5. Campaign websites and official candidate pages.
6. Public social media only when relevant to official conduct, campaign activity, or school governance.

## Hard Guardrails

- No unsourced accusations.
- No scraped private photos.
- No private student or minor information.
- No facts editable by a claimant.
- No suppression of constitutionally protected comments based only on viewpoint.
- Anonymous comments are visible when lawful and policy-compliant, but receive lower trust weight than verified official, parent, journalist, researcher, or voter accounts.
